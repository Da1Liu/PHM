# ADR-006 首台真机落地主系统 = 主轴

状态: 已采纳 (2026-06 整合期, 推翻"液压先行")

## Decision
首台真机第一个落地的 PHM 系统 = **主轴** (非液压)。

## Reason
A2 据 OPCUA地址对照.xlsx 填 signal 维表 (41 信号) 时发现:
- **该机液压只有 bool 压力监测, 无连续压力/流量** → 做不了 UCI 式液压效率基线, 液压退化为 L1 状态标志位。
- 成对轴 X1/X2·Y1/Y2·B1/B2 **共用同一 OPC UA 地址** (读到相同值) → 进给做基线前必须 probe 实测确认。
- **主轴信号最全**: 振动 + 轴承温度(耦合) + 电机温度(混淆) + 转速/档位(工况) + 电流。

## Consequence
- "液压第一个落地"这一说法**仅指算法验证顺序** (UCI 液压数据集是最接近已验证的场景), 不等于首台真机的落地顺序。
- 进给做基线前**必须 probe 实测**确认共址轴是否冗余。
- 旧文档 (`PROJECT_STATUS_AND_HANDOFF.md` 决策表、早期 memory) 的"液压先行"已加注澄清; 以本 ADR + `docs/CURRENT_STATE.md` 为准。
